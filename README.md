

## Run Locally

### 1. Clone repo

```
$ git clone git@github.com:yba73/Buccaneer-in-Gastronomy.git
```

### 2. Install MongoDB

Download it from here: https://docs.mongodb.com/manual/administration/install-community/

### 3. Run Buccaneer in Gastronomy

```
$ npm install
$ npm run dev
```

### 4. Run client

```
# open new terminal
$ npm install
$ npm start
```
